=== How to use ===

== For Windows x86 / Windows x64 + Java x86

.\realmote.cmd -remote COM:COM0 -roomba btspp://deadbeaf

== For Windows x64 + Java x64

.\realmote.cmd -remote COM:COM0 -roomba btspp://deadbeaf

== For Mac OS X

./realmote.sh -remote /dev/tty0 -roomba btspp://deadbeaf
